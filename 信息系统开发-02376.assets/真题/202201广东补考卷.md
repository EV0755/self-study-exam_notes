2. 使用结构化语言说明以下关系 5.3.1 结构化语言

   1. 小于19岁 

      1. 初中文化
         1. 脱产学习
      2. 高中文化
         1. 电工

   2. 大于19 小于45

      1. 初中文化或高中文化
         1. 男
            1. 钳工
         2. 女
            1. 车工
      2. 大学文化
         1. 技术员

   3. 大于45岁

      1. 技术员

   4. 答案：

      ```basic
      READ 年龄
      READ 文化水平
      READ 性别
      SELECT CASE
      CASE 1(年龄<19)
      	BEGIN IF
      		IF 文化水平=初中文化
      			脱产学习
      		ELSE 文化水平=高中
      			电工
        END IF
      CASE 2(19<年龄<45)
        BEGIN IF
          IF (文化水平=初中文化) 或 (化水平=高中)
          	BEGIN IF
          		IF (性别=男)
          			钳工
          		ELSE (性别=女)
          			车工
          ELSE 文化水平=大学
          	技术员
        END IF
      CASE 3(45<年龄)
      	技术员
      ```
